# AI-Powered Attack Surface Management Platform

## Overview

AI-Powered Attack Surface Management (ASM) Platform is a cybersecurity tool designed to help identify and analyze an organization's external attack surface.

The platform uses Google Gemini 2.5 Flash to provide AI-assisted security insights, attack scenario generation, risk analysis, and remediation recommendations.

This project demonstrates concepts related to Attack Surface Management, OSINT, risk assessment, and AI-assisted cybersecurity analysis.

---

## Features

- Asset Discovery Analysis
- Attack Surface Visualization
- Risk Assessment
- AI-Powered Security Recommendations
- Attack Scenario Generation
- Security Report Generation
- Interactive User Interface

---

## Technologies Used

- HTML
- CSS
- JavaScript
- Google Gemini 2.5 Flash API

---

## How to Use

1. Download or clone the repository.
2. Open `AI_powered_asm_platform.html` in a web browser.
3. Enter your Google Gemini API Key.
4. Start the security assessment.
5. Review generated analysis and recommendations.
